Online Railway Reservation System
This is a simple command-line application for booking and managing railway tickets. It's built using Java and consists of three main classes: TicketBooker, Passenger, and Main.

Features inculcated is listed below:
Book Ticket: Allows users to book railway tickets by providing passenger details and choosing their preferred seats.
Cancel Ticket: Allows users to cancel their booked tickets, making those seats available for others.
Available Tickets: Displays the number of available tickets for each train.
Booked Tickets: Shows a list of tickets that have been booked.
Exit: Exits the application.
